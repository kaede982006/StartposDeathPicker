Startpos Death Picker is built around one idea: death data should stay useful without getting in the way of playing.

It records total deaths for each level and breaks them down by 1% progress buckets from 0% to 99%. If a level uses start positions, each start position gets its own death history so practice trouble spots are easier to see.

The tracker keeps data per level, saves it to disk, and adjusts existing records when start positions are added, removed, or moved.

**Key Controls**

**`D`**: Open or close Startpos Death Picker before entering a level.

**`T`**: Toggle between bar and line graph views while the popup is open.

**`H` / `L`**: Move left or right across the tracked level and start position sections while the popup is open.
